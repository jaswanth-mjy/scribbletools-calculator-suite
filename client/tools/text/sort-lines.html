Sort Lines

  • Sort A-Z: Arrange lines in ascending (alphabetical) order. Useful for alphabetizing names, items, or code blocks.
  • Sort Z-A: Arrange lines in descending (reverse alphabetical) order. Great for finding the last entries or reversing sorted lists.
  • Shuffle: Randomly mix the order of lines. Perfect for random draws, shuffling quiz questions, or creating randomized lists.

How to use:

  1. Paste or type your multi-line text into the input box below.
  2. Click Sort A-Z, Sort Z-A, or Shuffle to process your lines.
  3. View the result instantly in the output area.
  4. Use Copy to copy the result, Download to save as a file, or Clear to reset.

Example uses: Alphabetize a guest list, sort log entries, randomize quiz questions, organize code imports, and more.
